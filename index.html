  • Plateforme Puzzle

Projet personnel réalisé sous Unity. Jeu vidéo de plateforme réalisé à partir d’un cours provenant du site OpenClassroom.

  • Ordre de Mission

Projet de deuxième année de DUT Informatique. Site web administratif pour l’IUT. Première expérience méthode agile et travail en équipe avec 6 personnes.

Créer un ordre de mission n’est pas disponible pour les élèves de l’IUT du campus III, ce qui est dangereux pour les missions tel que le salon de l’étudiant, car on n’aurait pas de preuve pour l’assurance. Ce site vient combler ce problème.
